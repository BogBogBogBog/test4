Minecraft Edition

Minecraft Edition คือฉบับของเกมที่สร้างขึ้นด้วยภาษาโปรแกรมที่ต่างกันโดยมีหลักๆอยู่ 2 Editions คือ Java Edition และ Bedrock Edition โดยทั้ง 2 Editions นี้มีความแตกต่างกัน ดังนี้

Java Edition

  • การเล่นข้ามแพลตฟอร์ม : เล่นกับเพื่อนๆ ที่เล่น Java Edition บน Windows, Mac และ Linux
  • ปรับแต่งสกินของคุณ : คุณสามารถดาวน์โหลดสกินจากอินเทอร์เน็ตหรือสร้างสกินของตัวเองแล้วอัปโหลดมาใช้ในเกม
  • ปรับแต่งด้วยม็อด : ดาวน์โหลดม็อดจากชุมชนเพื่อปรับเปลี่ยนประสบการณ์การเล่นของคุณ
  • เข้าร่วม Realm : Realms คือเซิร์ฟเวอร์ที่ดำเนินการโดย Minecraft ที่สำหรับคุณและเพื่อนๆ เชิญเพื่อนเข้าร่วมได้ไม่จำกัดจำนวนและเล่นกับเพื่อน 10 คนในเวลาเดียวกัน
  • โฮสต์ Realm : แชร์โลกของคุณกับเพื่อนๆ และผู้เล่นจากทั่วโลก! โฮสต์เซิร์ฟเวอร์เกมหลายผู้เล่นของคุณเอง และให้เพื่อนๆ เข้าร่วมจากมือถือและ PC
  • เข้าร่วมสังคมผู้เล่น : มีสังคมผู้เล่นมากมายที่ทุ่มเทในการปรับแต่ง Java Edition ด้วยม็อดต่างๆ เพื่อเปลี่ยนประสบการณ์การเล่นหลายผู้เล่น

Bedrock Edition

  • การเล่นข้ามแพลตฟอร์ม: เล่นกับเพื่อนๆ ที่เล่น Bedrock Edition บน Windows 10, Windows 11, Xbox, Nintendo Switch, PS4 และมือถือ
  • รองรับการควบคุมด้วยคอนโทรลเลอร์: Minecraft: Bedrock Edition มีตัวเลือกให้เปลี่ยนไปใช้คอนโทรลเลอร์สำหรับการเคลื่อนไหวหรือควบคุมสัมผัสหากเล่นบนอุปกรณ์มือถือ
  • ซื้อจาก Minecraft Marketplace: สามารถซื้อแพ็คจากตลาดเพื่อเปลี่ยนประสบการณ์การเล่นของคุณ เนื้อหาที่ดาวน์โหลดได้: ใช้ Add-ons เพื่อปรับแต่งประสบการณ์ของคุณ เปลี่ยนรูปลักษณ์ และแม้กระทั่งเปลี่ยนพฤติกรรมของม็อบ
  • เซิร์ฟเวอร์หลายผู้เล่น: เซิร์ฟเวอร์สาธารณะที่เป็นพันธมิตรกับ Minecraft ให้ทุกคนเข้าร่วมได้ รวมถึงเกมมินิเกมส์หลายคน
  • เข้าร่วม Realm: Realms คือเซิร์ฟเวอร์ที่ดำเนินการโดย Minecraft ที่สำหรับคุณและเพื่อนๆ เชิญเพื่อนเข้าร่วมได้ไม่จำกัดจำนวนและเล่นกับเพื่อน 10 คนในเวลาเดียวกัน
  • การแชร์ในครอบครัว: คุณสามารถแชร์ Minecraft: Bedrock Edition ให้กับคนในครอบครัวโดยใช้ Family Share บน PC
  • โฮสต์ Realm: แชร์โลกของคุณกับเพื่อนๆ และผู้เล่นจากทั่วโลก! โฮสต์เซิร์ฟเวอร์เกมหลายผู้เล่นของคุณเอง และให้เพื่อนๆ เข้าร่วมจากมือถือและ PC
  • ความสำเร็จและถ้วยรางวัล: ทำภารกิจบางอย่างให้สำเร็จเพื่อรับความสำเร็จหรือถ้วยรางวัลในโหมดการเอาชีวิตรอด
  • การควบคุมและการควบคุมสำหรับผู้ปกครอง: เชื่อมต่อกับแอป Xbox Family Settings เพื่อปรับแต่งประสบการณ์การเล่นสำหรับคุณและครอบครัว
  • การแชร์ในครอบครัว: คุณสามารถแชร์ Minecraft: Bedrock Edition ให้กับคนในครอบครัวโดยใช้ Family Share บน PC

ยังมีความแตกต่างอื่นๆ อีกมากมายระหว่างสอง Edition นี้ ฮย่างเช่น ประสบการณ์การเล่น Gameplay และรายละเอียดของเกมเล็กน้อย เป็นต้น
